# XXQ VCU - 整车控制器

基于 **STM32F105VC** 的整车控制器 (VCU) 固件项目，使用 STM32CubeMX + Keil MDK-ARM 开发。

## 硬件平台

- **MCU**: STM32F105VC (Cortex-M3, 72MHz)
- **CAN**: 双路 CAN (CAN1 + CAN2)，支持环回测试模式
- **ADC**: 多通道模拟量采集（DMA 连续采样）
- **UART**: USART1 遥测上报
- **GPIO**: LED 指示、蜂鸣器、制动灯、档位开关等

## 功能

| 模块 | 说明 |
|------|------|
| **档位切换** | P→D 切换，含刹车踏板互锁逻辑（必须先踩刹车再按开关） |
| **制动灯控制** | 踏板位置检测，滞回区间防抖（开 >5%, 关 <2%） |
| **蜂鸣器** | 进入 D 档确认音，3 秒后自动关闭 |
| **LED 指示** | LED1 1Hz 闪烁，系统运行指示 |
| **传感器采集** | 电池电压、油门、刹车、转向角度 (ADC + DMA) |
| **CAN 通信** | 双 CAN 初始化，环回自收发测试 |
| **遥测上报** | UART 每秒输出传感器数据及档位状态 |

## 项目结构

```
├── Core/
│   ├── Inc/          # 头文件 (vehicle, can, adc, gpio 等)
│   └── Src/          # 源文件 (main, vehicle, can, adc 等)
├── Drivers/          # HAL 驱动库
├── MDK-ARM/          # Keil MDK 工程文件
├── build/            # 编译输出
└── XXQ_VCU.ioc       # STM32CubeMX 配置文件
```

## 主循环调度

```
每次循环:  档位切换检测 → 制动灯控制
500ms:    LED1 闪烁
1000ms:   档位状态计时 → UART 遥测 → CAN 测试发送
```

## 构建

使用 Keil MDK-ARM 打开 `MDK-ARM/XXQ_VCU.uvprojx`，编译下载。

或通过 STM32CubeMX 重新生成代码后构建：
1. 打开 `XXQ_VCU.ioc`
2. 生成代码
3. 编译烧录
